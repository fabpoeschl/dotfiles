NAVIGATION                                           *cheatsheet-navigation*

Movement ~
  j / k             Move by visual line (not physical line)
  B                 Beginning of line (like ^)
  E                 End of line (like $)
  gV                Highlight last inserted text
  <C-h/j/k/l>      Navigate between windows

Jump (flash.nvim) ~
  s                 Flash jump — type 2 chars, pick a label
  S                 Flash treesitter — select treesitter nodes
  r                 Remote flash (operator-pending)

Search ~
  <leader><space>   Clear search highlighting
  K                 Grep word under cursor -> quickfix
  <leader>fg        Live grep (Telescope)
  <leader>fs        Document symbols (Telescope)
  <leader>fd        Diagnostics (Telescope)
  <leader>ft        Find TODOs (Telescope)

EDITING                                              *cheatsheet-editing*

Basic ~
  <leader>w         Save file (:w!)
  <leader>q         Quit window
  <leader>S         Strip trailing whitespace
  <leader>n         Toggle relative line numbers
  <space>           Toggle fold

Clipboard ~
  <leader>y         Yank to system clipboard
  <leader>x         Cut to system clipboard
  <leader>p         Paste from system clipboard
  <C-c>             Copy to system clipboard (normal & visual)

Visual Mode ~
  J                 Move selected lines down
  K                 Move selected lines up
  <                 Indent left (keeps selection)
  >                 Indent right (keeps selection)

Comment (Comment.nvim) ~
  gcc               Toggle comment on line
  gc                Toggle comment on selection (visual)

Surround (nvim-surround) ~
  ys{motion}{char}  Add surround         e.g. ysiw" → surround word with "
  ds{char}          Delete surround       e.g. ds" → remove surrounding "
  cs{old}{new}      Change surround       e.g. cs"' → change " to '
  S{char}           Surround selection (visual mode)

Auto-Format (conform.nvim) ~
  <leader>cf        Format buffer manually
  (auto)            Files are formatted on save automatically

Window Resizing ~
  <C-Up>            Increase window height
  <C-Down>          Decrease window height
  <C-Left>          Decrease window width
  <C-Right>         Increase window width

DEBUGGER (nvim-dap)                                    *cheatsheet-dap*

Stepping ~
  <F5>              Continue / start debug session
  <F10>             Step over
  <F11>             Step into
  <F12>             Step out

Breakpoints ~
  <leader>db        Toggle breakpoint
  <leader>dB        Conditional breakpoint (prompts for expression)
  <leader>dl        Log point (prints message without stopping)

Session Control ~
  <leader>dx        Continue
  <leader>dr        Restart session
  <leader>dq        Terminate session
  <leader>dp        Pause execution
  <leader>dC        Run to cursor

UI ~
  <leader>di        Toggle DAP UI panels
  <leader>de        Evaluate expression (works in visual mode too)
  <leader>dh        Hover variable under cursor
  <leader>do        Toggle REPL

Debug Configurations ~

  Python:
    Launch file                         Run current .py file
    Launch with arguments               Prompts for CLI arguments
    Attach remote (localhost:5678)      For kubectl port-forward / debugpy
    Attach remote (custom host:port)    Prompts for host and port

  JavaScript / TypeScript:
    Launch file (Node)                  Run current file with Node
    Attach to Node (localhost:9229)     For --inspect / kubectl port-forward
    Attach to Node (custom host:port)   Prompts for host and port
    Launch Chrome                       Debug in Chrome (prompts for URL)

  Ruby:
    Launch file                         Run current .rb file
    Launch with bundler                 bundle exec ruby <file>
    Launch Rails server                 bundle exec rails server
    Launch RSpec (current file)         bundle exec rspec <file>
    Attach remote (localhost:1234)      For rdbg / kubectl port-forward
    Attach remote (custom host:port)    Prompts for host and port

Remote Debugging (Kubernetes) ~

  The "Attach remote" configurations connect to a debug server over TCP.
  Use kubectl port-forward to bridge the pod's debug port to localhost:

    # Python (debugpy)
    kubectl port-forward pod/my-app 5678:5678
    # Then: <F5> → "Attach remote (localhost:5678)"

    # JavaScript / Node (--inspect)
    kubectl port-forward pod/my-app 9229:9229
    # Then: <F5> → "Attach to Node (localhost:9229)"

    # Ruby (rdbg / debug.gem)
    kubectl port-forward pod/my-app 1234:1234
    # Then: <F5> → "Attach remote (localhost:1234)"

  On the pod, start the app with its debug server enabled:

    # Python:  python -m debugpy --listen 0.0.0.0:5678 --wait-for-client app.py
    # Node:    node --inspect=0.0.0.0:9229 app.js
    # Ruby:    rdbg --open --host 0.0.0.0 --port 1234 -c -- ruby app.rb

DATABASE (vim-dadbod)                                *cheatsheet-database*

Connection Management ~
  <leader>du        Toggle DBUI drawer
  <leader>da        Add a connection manually
  <leader>dc        Connect to remote DB (K8s pod)
  <leader>dd        Disconnect remote DB

Remote Connect (:DBConnect) ~
  :DBConnect -c <ctx> -n <ns> -d <db>     Connect with defaults
  :DBConnect -c <ctx> -n <ns> -d <db> -p 5433 -b mydb -u admin

  Flags:
    -c, --context     Kubernetes context
    -n, --namespace   Kubernetes namespace
    -d, --database    Pod name substring to match
    -s, --secret      Secret name (defaults to -d value)
    -k, --key         Secret key (default: password)
    -u, --user        DB user (default: postgres)
    -p, --port        Remote port (auto-detected if omitted)
    -b, --dbname      Database name in URL (defaults to -d value)

  :DBDisconnect       Stop port-forward and remove connection

Local Databases ~
  Use <leader>da to add a local connection URL directly:
    postgres://user:pass@localhost:5432/mydb
    mysql://root@localhost:3306/mydb
    sqlite:path/to/db.sqlite

SQL Completion ~
  Autocompletion for table and column names is active in .sql files
  when a dadbod connection is configured.

KUBERNETES                                        *cheatsheet-kubernetes*

Pod Connection (:PodConnect) ~
  <leader>ks        Shell into pod (prompts for context/ns/app)
  <leader>kl        Tail pod logs
  <leader>kf        Port-forward pod

  :PodConnect -c <ctx> -n <ns> -a <app>                 Shell (default)
  :PodConnect -c <ctx> -n <ns> -a <app> logs             Tail logs
  :PodConnect -c <ctx> -n <ns> -a <app> forward 3000:3000

  All actions open in a terminal split. Close the buffer to stop.

Shell Scripts (~/.local/bin/) ~
  remote-db-connect -c <ctx> -n <ns> -d <db> [-p <port>]
  remote-pod-connect -c <ctx> -n <ns> -a <app> [shell|forward|logs]
